<img src="/public/images/bibril-og.png" alt="og" />

## Installation

1. Clone the repo:

```bash
git clone https://github.com/BadreddineIbril/bibril.git
```

2. Go into the project folder:

```bash
cd bibril
```

3. Install dependencies:

```bash
bun install
```

4. Start the development server:

```bash
bun run dev
```
